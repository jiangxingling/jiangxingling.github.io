使用Github+Hexo搭建个人博客

发表于 2017-04-17

一、准备工作

  1. Git Bash
  2. 账号
  3. node.js
    到node官网上下载安装即可

二、安装Hexo

  1. npm 默认的源的下载速度可能很慢，所以使用淘宝镜像替换。执行下面的命令，将npm的源设置成淘宝镜像站：

    npm config set registry “https://registry.npm.taobao.org“

  2. 然后执行如下命令，安装hexo

    npm install -g hexo-cli

三、Hexo本地设置

  1. 在E盘下新建Blog文件夹，初始化Hexo

    cd Blog
    hexo init

  2. 安装 npm

    npm install

  3. 开启hexo服务器

    hexo s (或 hexo server)

  4. 打开http://localhost:4000/,可以访问HEXO页面，则本地已配置完成

四、Hexo关联github

  1. 在github新建repository,名为jiangxingling.github.io固定写法
  2. 在Blog目录下修改_config.yml

==注意：所有的”:”后面都需要加上空格==

  1. 执行生成静态页面的命令

    hexo generate 或者：hexo g

此时若出现如下报错：

ERROR Local hexo not found in ~/blog
ERROR Try runing: ‘npm install hexo –save’

则执行命令：

npm install hexo –save

若无报错，则忽略此步骤

  1. 执行配置命令

    hexo deploy 或者：hexo d

若执行命令hexo deploy仍然报错：无法连接git或找不到git，则执行如下命令来安装hexo-deployer-git

npm install hexo-deployer-git –save

  1. 打开https://jiangxingling.github.io/,能够正常访问则完成初始化
  2. 添加ssh key到github

五、发布文章

  1. 执行如下命令新建文章

    hexo new “name.md”

名为”name.md”的文件在/blog/source/_posts/目录下

  1. 编辑文章后发表

hexo generate //生成静态页面

hexo deploy //将文章部署到github

六、改变Theme

  1. 在Hexo官网主题页去搜寻自己喜欢的theme，以NexT主题为例。进入Blog目录执行如下命令:

git clone https://github.com/iissnan/hexo-theme-next themes/next

或者是下载主题，将解压文件夹放到themes目录下

  1. 将blog目录下_config.yml里theme的名称landscape修改为next
  2. 到blog目录下依次执行如下命令(每次部署文章的步骤)

七、常见问题解决

1. 每次hexo d都要重新输入账户密码

将_config.yml文件中配置的repository: https://github.com/jiangxingling/jiangxingling.github.io 改为 git@github.com:jiangxingling/jiangxingling.github.io
