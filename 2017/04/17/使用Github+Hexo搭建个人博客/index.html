使用Github+Hexo搭建个人博客

发表于 2017-04-17

准备工作

  1. Git Bash
  2. github账号
  3. node.js
    到node官网上下载安装即可

安装Hexo

  1. npm 默认的源的下载速度可能很慢，所以使用淘宝镜像替换。执行下面的命令，将npm的源设置成淘宝镜像站：

    npm config set registry “https://registry.npm.taobao.org“

  2. 执行如下命令，安装hexo

    npm install -g hexo-cli

Hexo本地设置

  1. 在E盘下新建Blog文件夹，初始化Hexo

    cd Blog
    hexo init

  2. 安装 npm

    npm install

  3. 开启hexo服务器

    hexo s (或 hexo server)

  4. 打开http://localhost:4000/,可以访问hexo页面，则本地已配置完成

发布文章

  1. 执行如下命令新建文章
    hexo new “name”
    名为”name.md”的文件在/blog/source/_posts/目录下
  2. 编辑文章后发表

改变Theme

  1. 在Hexo官网主题页去搜寻自己喜欢的theme，以NexT主题为例。进入Blog目录执行如下命令:

    git clone https://github.com/iissnan/hexo-theme-next themes/next
    或者是下载主题，将解压文件夹放到themes目录下
